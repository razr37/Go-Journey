# Go Journey

A discovery-first Go learning app that teaches beginners through interactive board examples, Bobby Fischer-style micro-puzzles, and a local AI Coach scaffold.

## Deploy

This is a Render static site. Publish the repository root.

## Current version

V1.4 adds Puzzle Book mode, an 8-puzzle Fischer sprint, expanded quick-fire 9x9 to 19x19 puzzles, and AI Coach v1 with mistake tracking and targeted drills.


## V1.6 Full Dan Path
- Puzzle Book now contains 150 puzzles total.
- First 50 are intermediate bridge puzzles.
- Final 100 are dan-track Fischer puzzles covering haengma, joseki judgment, ladder reading, ko threats, tesuji, sabaki, fuseki, reduction/invasion, and 19x19 direction of play.
- AI Coach v1.6 keeps the rule-based diagnostic layer and recommends review from missed pattern families.
